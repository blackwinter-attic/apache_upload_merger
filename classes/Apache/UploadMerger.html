Public Class methods

new(map = {})

Creates a new RubyHandler instance for the Apache web server. It is to be installed as a custom 404 ErrorDocument handler.

The argument map contains key/value pairs of URL prefixes and upload base directories.

handler(request)

If the current request asked for a resource that‘s not there, it will be copied from one of the appropriate upload directories, determined by its URL prefix. Otherwise, the original error will be thrown.
